**2. Create `src/lib/jsonld.ts`** -- JSON-LD builder functions:

Install schema-dts first: `npm install schema-dts`

```typescript
import type { WithContext, HomeAndConstructionBusiness, Service } from 'schema-dts'
import { SITE_CONFIG } from './metadata'
```

Create two exported functions:

`generateLocalBusinessJsonLd()` -- returns `WithContext<HomeAndConstructionBusiness>` object:
- `@context: 'https://schema.org'`
- `@type: 'HomeAndConstructionBusiness'`
- `name: SITE_CONFIG.name`
- `description: SITE_CONFIG.description`
- `url: SITE_CONFIG.url`
- `telephone: '+1-555-123-4567'` with `// PLACEHOLDER -- replace with confirmed NAP from owner`
- `address` with PostalAddress type containing placeholder street, city (Hamilton), province (ON), country (CA), postal code -- ALL marked with PLACEHOLDER comments
- `areaServed: { '@type': 'AdministrativeArea', name: 'Southern Ontario' }`
- `makesOffer` array with 4 Offer items, each containing a Service with name and description matching the SERVICES data

`generateServicePageJsonLd()` -- returns `WithContext<Service>[]` or a combined structure for the services page. Can reuse makesOffer data but with the expanded descriptions from SERVICES_EXPANDED.

Both functions return plain objects (not JSX). Consumers call `JSON.stringify()`.
