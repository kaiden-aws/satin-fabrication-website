**src/lib/constants.ts** — Export `NAV_LINKS` array with 4 entries: `{ label: 'WORK', href: '#work' }`, `{ label: 'SERVICES', href: '#services' }`, `{ label: 'PROCESS', href: '#process' }`, `{ label: 'CONTACT', href: '#contact' }` typed as `as const`. Also export `SCROLL_THRESHOLD = 50` and `CURSOR_LERP_SPEED = 0.15`.

**src/hooks/useScrolled.ts** — `'use client'` hook. Uses `useScroll` from `motion/react` to get `scrollY`, uses `useMotionValueEvent` to listen for changes and `useState` to toggle a boolean when scroll passes `threshold` parameter (default 50). Returns the boolean. This is the pattern from the research — do NOT use `window.addEventListener('scroll')`.

**src/components/layout/SkipLink.tsx** — Server component (no `'use client'`). Renders an `<a href="#main-content">` that is `sr-only` by default and becomes visible + styled on focus: `focus:not-sr-only focus:fixed focus:top-4 focus:left-4 focus:z-[9999] focus:px-4 focus:py-2 focus:bg-void focus:text-gold focus:border focus:border-gold focus:rounded focus:outline-none focus:ring-2 focus:ring-gold font-body text-sm`. Text: "Skip to content".

**src/components/layout/NavLink.tsx** — Server component. Accepts `{ href: string, label: string, className?: string }`. Renders `<a>` with `relative font-body text-sm uppercase tracking-widest text-cream transition-colors duration-300 hover:text-gold group` and a child `<span>` for the underline: `absolute bottom-0 left-0 h-[1px] w-full bg-gold origin-left scale-x-0 transition-transform duration-300 ease-luxury group-hover:scale-x-100`. This is pure CSS animation — no JS needed. Uses `cn()` from `@/lib/utils`. The `ease-luxury` class maps to the `--ease-luxury` token.

**src/components/layout/HamburgerButton.tsx** — `'use client'` component. Accepts `{ isOpen: boolean, toggle: () => void }`. Renders a `<button>` with `aria-expanded={isOpen}` and `aria-label`. Contains an SVG (24x24 viewBox) with two `m.path` elements from `motion/react`. Uses variants: top line `closed: { d: 'M4 8L20 8' }` / `open: { d: 'M4 4L20 20' }`, bottom line `closed: { d: 'M4 16L20 16' }` / `open: { d: 'M4 20L20 4' }`. Transition uses luxury easing `[0.25, 0.1, 0.25, 1]`, duration 0.3s. Button is `md:hidden` (only visible on mobile), z-[70] so it sits above the mobile menu overlay. Stroke color is `currentColor` (cream), strokeWidth 1.5, strokeLinecap "round".

**src/components/layout/MobileMenu.tsx** — `'use client'` component. Accepts `{ isOpen: boolean, onClose: () => void }`. Uses `AnimatePresence` from `motion/react`. When `isOpen`, renders a `m.div` overlay at `fixed inset-0 z-[60] bg-void flex flex-col items-center justify-center` with `role="dialog" aria-modal="true" aria-label="Navigation menu"`. The overlay uses `overlayVariants` (closed: opacity 0, open: opacity 1). Inside, a `m.ul` with `listVariants` using `staggerChildren: 0.1, delayChildren: 0.15`. Each `m.li` uses `itemVariants` (closed: opacity 0, y: 20; open: opacity 1, y: 0 with luxury easing, duration 0.4s). Links render from `NAV_LINKS` imported from `@/lib/constants`. Each link uses `font-display text-4xl text-cream hover:text-gold transition-colors` and calls `onClose` on click. CRITICAL: Include a `useEffect` that locks body scroll when open (`document.body.style.overflow = 'hidden'`) and applies `inert` attribute to `#main-content` element. Restore both on close and in cleanup. The child of `AnimatePresence` must have a `key` prop (use `key="mobile-menu"`). Define `exit` animations matching the closed variants.

**src/components/layout/Navbar.tsx** — `'use client'` component. Uses `useScrolled` hook to detect scroll past threshold. Contains `useState` for `mobileMenuOpen`. Renders `m.nav` at `fixed top-0 left-0 right-0 z-40 px-6 py-4` with `initial={false}` (CRITICAL: prevents flash on mount per Pitfall 1). Animate background: when scrolled, `backgroundColor: 'rgba(10, 10, 10, 0.95)'` with `backdrop-blur-md` class; when not scrolled, `backgroundColor: 'rgba(10, 10, 10, 0)'`. Transition duration 0.3s. Inside the nav: a flex container with justify-between and items-center. Left side: `<a href="/">` with "SATIN" in `font-display text-xl text-gold tracking-wider`. Right side (desktop): `NAV_LINKS.map` rendering `NavLink` components, wrapped in a `hidden md:flex gap-8 items-center` container. Right side (mobile): `HamburgerButton` with `isOpen={mobileMenuOpen}` and `toggle={() => setMobileMenuOpen(prev => !prev)}`. Render `MobileMenu` with `isOpen={mobileMenuOpen}` and `onClose={() => setMobileMenuOpen(false)}`.

IMPORTANT: Use `m` (not `motion`) from `motion/react` everywhere — LazyMotion strict mode will throw if `motion.*` is used.
