  <action>
Create two client components in src/components/ui/ (create the ui/ directory first).

**MotionWrapper.tsx** — Scroll-triggered reveal wrapper:
- 'use client' directive
- Import { m } from 'motion/react' (NOT motion.* — LazyMotion strict mode)
- Props: children (ReactNode), delay (number, default 0), className (string, optional), as (keyof JSX.IntrinsicElements, optional, default 'div') for semantic HTML support
- Use m.div with:
  - initial={{ opacity: 0, y: 30 }}
  - whileInView={{ opacity: 1, y: 0 }}
  - viewport={{ once: true, margin: '-100px' }} — triggers slightly before fully visible
  - transition={{ duration: 0.7, delay, ease: [0.25, 0.1, 0.25, 1] }} — matches DSGN-04 luxury easing
- MotionConfig reducedMotion="user" from Providers automatically suppresses animation — no extra code needed
- Export as named export

**ParallaxWrapper.tsx** — Scroll-speed parallax wrapper:
- 'use client' directive
- Import { m, useScroll, useTransform } from 'motion/react'
- Import { useRef } from 'react'
- Props: children (ReactNode), speed (number, default 0.5), className (string, optional)
- Create a ref for the container div
- useScroll with target: ref, offset: ['start end', 'end start'] — tracks full element traversal through viewport
- Compute yOffset from speed: const yOffset = (1 - speed) * 200 — pixels of travel. speed < 1 moves slower (background parallax), speed > 1 moves faster
- useTransform(scrollYProgress, [0, 1], [-yOffset, yOffset]) to create y motion value
- Render m.div with ref and style={{ y }}
- Motion values applied via style prop bypass React re-renders for 60fps
- Export as named export

Both files must use m.* components only (not motion.*) per LazyMotion strict mode.
