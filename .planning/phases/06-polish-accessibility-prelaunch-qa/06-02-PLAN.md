**PortfolioSection.tsx — Touch-friendly overlay:**

Replace the current overlay div's class from:
```
opacity-0 group-hover:opacity-100
```
to:
```
opacity-100 [@media(hover:hover)]:opacity-0 [@media(hover:hover)]:group-hover:opacity-100
```

This means:
- On touch devices (no hover capability): overlay is always visible (opacity-100). The gradient overlay provides text readability over the image.
- On hover-capable devices: overlay starts hidden (opacity-0) and appears on hover (group-hover:opacity-100) — preserving existing desktop behavior.

The full overlay div class becomes:
```
absolute inset-0 flex flex-col justify-end p-4 bg-gradient-to-t from-black/80 via-black/40 to-transparent opacity-100 [@media(hover:hover)]:opacity-0 [@media(hover:hover)]:group-hover:opacity-100 transition-opacity duration-300 ease-[cubic-bezier(0.25,0.1,0.25,1)]
```
