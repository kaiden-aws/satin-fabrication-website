- In the existing `@media (prefers-reduced-motion: reduce)` block, the current rule sets `animation-duration: 0.01ms !important` which causes a single flash frame.
- Add a specific rule to fully remove the hero gradient animation:
```css
@media (prefers-reduced-motion: reduce) {
  *,
  *::before,
  *::after {
    animation-duration: 0.01ms !important;
    animation-iteration-count: 1 !important;
    transition-duration: 0.01ms !important;
    animation-play-state: paused !important;
  }
}
```
- Adding `animation-play-state: paused !important` ensures CSS animations are frozen on their first frame rather than briefly executing.
